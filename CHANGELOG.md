### 寻求帮助：超星学习通乱码解码方案

当前系统对超星导出的乱码文本采用**字符级映射表**进行批量替换（见 `clean.py` 中的 `CHAR_MAP`），这是一种暴力穷举的补救方式，并非真正的编码解码方案。

如果你了解以下任一方向，欢迎通过邮箱或微信联系作者，万分感谢！
- 超星学习通题库导出文件的原始编码格式
- 乱码的底层成因（GBK/UTF-8 混合编码、数据库直读等）
- 能从超星数据库或 API 直接提取无乱码文本的方法

提供有效解码方案或协助实现者，将在公告页和 README 中署名致谢。

## v2.2.1 (2026-06-24)

### 修复
- 修复「共同体概论」「大学语文」加载时报错 `Cannot read properties of undefined (reading 'label')` 的问题
